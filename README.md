# Tetris-Clone

This is tetris, cloned as a full featured game.
The version of the popular classic is made using python
and the panda3d game engine 1.8.1.

To run the code download and install panda3d as per the 
instructions on the panda3d website.
Then assuming you have got panda3d correctly setup, from the comand line
cd to the gamemanager.py file and run the file as one would any other panda3d app.

Controls are as follows
left:       A
right:      D
down:       S
rotate:     W
pause game: esc

The game also features an ingame media player.
